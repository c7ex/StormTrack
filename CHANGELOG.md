## [1.0.0] - 2026-05-08

### Added
- Первый публичный релиз StormTrack.
- Отображение нескольких графиков в одном окне.
- Потоковое обновление данных в реальном времени (стриминг).
- Поддержка статических данных с выборками до 1 миллиона точек и более.
- Масштабирование колёсиком мыши относительно курсора; раздельное масштабирование по X при зажатом Shift; быстрый зум с Ctrl.
- Панорамирование перетаскиванием левой кнопкой мыши.
- Автоматическая подстройка видимой области по оси X под все активные трейсы (клавиша A).
- Интерактивная легенда: скрытие/отображение трейсов кликом по цветному квадрату.
- Отслеживание ближайшей точки данных при наведении курсора с подсветкой и отображением координат.
- Адаптивная сетка с числовыми подписями, линии сетки с прозрачностью.
- Изменение размера области графика перетаскиванием границ рамки.
- Многопоточная архитектура: окно графика работает в отдельном потоке, не блокируя консоль.
- Полное отсутствие внешних зависимостей — только чистый WinAPI/GDI.
- Безопасное управление GDI-ресурсами через RAII (устранены утечки кистей, перьев, шрифтов, регионов).
- API переработано: класс `WindowObject` переименован в `StormTrack` с методами `StaticData`, `AddStreamingTrace`, `Streaming`.
- Отображение FPS в углу графика.
